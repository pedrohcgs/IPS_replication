"Covariate Distribution Balance via Propensity Scores"


This file includes the following folders:

-- Simulations: includes all relevant R files to reproduce the Monte Carlo simulation results in the article. The subfolder "stylized" reproduce simulations in Section 5.1 of the paper. The subfolder "401k" reproduces the simulations on Section 5.2 of the paper.

-- Application: includes all relevant R and data files to reproduce the results of the empirical application in Section 6 of the paper.

-- Codes: Contain R scripts used in both simulation and Application.

All results presented in the paper are computed with R 4.0 with Windows 11
